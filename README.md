# definiciones recursivas 

para los trabajos de deficiones recursivas usaremos la siquiente estructura:

```
(*valor de retorno*)
TIPODEDATO retorno

(*caso base*)
SI consion HACER
	retorno = VALOR DE RETORNO EN CASO BASE
SINO
	(*paso resursivo*)
	retorno = OPERACION CON LLAMADO RECURSIVO
FIN SI
```
